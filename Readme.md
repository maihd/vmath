# vmath - C vector (graphics - 3D) math libary

## Feature
1. Vector, matrix support. Others later.
2. C/C++ inline pure functions
3. Type generic functions (C11)
4. Multi-platform (MacOS, Android tested)
5. NEON/SSE support
6. C++ operators overloading
